Code Blocks

Inline code works just as expected with minimal css.

Code blocks on the other hand, not so much. I found this weird hack where you first wrap <code> in <pre>, then juggle the indents through css. Strange, but it seems to works just fine! 


      for i in range(3):
          print(i)

One can even highlight and copy the contents without absolutely destroying the indent, something that is becoming more rare as websites ever strive to become more fancy.
